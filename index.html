Кодування

Для побудови графіків ваше число повинне бути у диапозоні:

  • для двійкової системи счислення [00000000;11111111];
  • для вісімкової системи счислення [0;377];
  • для десятичної системи счислення [0;255];
  • для шістнадцяткової системи счислення [00;FF].
